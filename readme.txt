== Description ==

Prevents the visual editor in the admin page from automatically expanding the height to fit the content - the way Wordpress 3.9 was

Because I work with extremely long documents, this feature was more of a irritation for me. That's why this plugin came about

== Installation ==

1. Upload 'disable-editor-autofit' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all!
